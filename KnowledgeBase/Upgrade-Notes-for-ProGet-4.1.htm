Upgrading from a version older than 4.x? It is strongly recommended that you first review Upgrading from ProGet v2 and v3 before proceeding.

ProGet 4.1 is minor release with the following changes:

  • Feed replication
  • PowerShellGet Feed Type
  • Various bug fixes
  • New API

Upgrade Impact

Because the changes are additive, the upgrade risks are negligible: there is a minor database schema change, and you can mitigate this risk by backing-up your ProGet instance prior to upgrade.

Upgrade Process

ProGet may be upgraded by simply running the latest installer. The agents will automatically update when the service starts again.

Rollback

Because there are database changes, a rollback will require uninstalling ProGet, and then restoring your ProGet instance.
