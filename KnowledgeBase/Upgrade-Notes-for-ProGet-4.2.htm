Upgrading from a version older than 4.x? It is strongly recommended that you first review Upgrading from ProGet v2 and v3 before proceeding.

ProGet 4.2 is minor release with the following changes:

  • Amazon S3 and Azure Blog Package Stores
  • New Package Store Editor UI

Upgrade Impact

This was originally planned as a maintenance release, but we felt the addition of new package stores warranted a minor release. Because the changes are entirely additive and very minor, the upgrade risks are negligible; there were no database schema changes.

Upgrade Process

ProGet may be upgraded by simply running the latest installer.

Rollback

See restoring your ProGet instance for how to rollback.
