Upgrading from a version older than 4.x? It is strongly recommended that you first review Upgrading from ProGet v2 and v3 before proceeding.

ProGet 4.6 is minor release with the following changes:

  • Package Vulnerability Tracking (new feature)
  • Amazon and Azure Package Stores are now Extensions (source code)
  • Miscellaneous Bug Fixes (release notes)
  • New SDK

Upgrade Impact

Most changes are additive but notable changes are:

  • "Vulnerabilities" heading added to top-level navigation
  • Vulnerabilties_Assess permission added and applied to built-in Administer task
  • Installer: The ProGet installer now installs two separate services; one is the traditional ProGet service (INEDOPROGETSVC), and the new one (INEDOPROGETWEBSVC) will host the Integrated Web Server if enabled and IIS is not used
  • Installer: The ProGet installer automatically installs the Amazon, Azure, and VorSecurity extensions. Visit the Extensions page within ProGet or the Inedo Den to download other extensions.

Upgrade Process

ProGet may be upgraded by simply running the latest installer.

Rollback

See restoring your ProGet instance for how to rollback.
