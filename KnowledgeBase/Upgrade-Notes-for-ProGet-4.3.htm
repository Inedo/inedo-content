Upgrading from a version older than 4.x? It is strongly recommended that you first review Upgrading from ProGet v2 and v3 before proceeding.

ProGet 4.3 is minor release with the following changes:

  • Docker Feed Support
  • Package Filter Extensibility

Upgrade Impact

Since the changes are entirely additive and very minor, the upgrade risks are negligible; the only database schema changes involved the creation of new tables and increasing column widths.

Upgrade Process

ProGet may be upgraded by simply running the latest installer.

Rollback

See restoring your ProGet instance for how to rollback.
