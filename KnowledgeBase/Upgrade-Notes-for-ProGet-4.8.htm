Upgrading from a version older than 4.x? It is strongly recommended that you first review Upgrading from ProGet v2 and v3 before proceeding.

ProGet 4.8 is minor release with the following changes:

  • Asset Directories (new feature)
  • Enhanced LDAP directory (new feature)
  • Miscellaneous Bug Fixes (release notes)
  • New SDK

Upgrade Impact

Most changes are additive but notable changes are:

  • "Assets" heading added to top-level navigation
  • Nonfunctional changes to the package storage subsystem due to new Asset Directory support
  • New Amazon and Azure extensions required (included) to support asset storage on these services
  • Feed cleanup task will no longer attempt to reindex packages if they have been deleted/edited outside of ProGet

Upgrade Process

ProGet may be upgraded by simply running the latest installer.

Rollback

See restoring your ProGet instance for how to rollback.
