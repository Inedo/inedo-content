Upgrading from a version older than 4.x? It is strongly recommended that you first review Upgrading from ProGet v2 and v3 before proceeding.

ProGet 4.4 is minor release with the following changes:

  • Package Promotion Support for Universal, NuGet, PowerShell, Chocolatey, and npm Feeds
  • Package Deployment Support for Universal, NuGet, PowerShell, and Chocolatey Feeds

Upgrade Impact

The feed view/search pages and package version pages for all feed types have been combined, therefore some feed-specific UI functionality may be missing or perform differently than in previous versions. Let us know if you encounter any oddities with these or other pages during your usage.

Other than the UI, new functionality since v4.3 is additive and upgrade risks are negligable.

Upgrade Process

ProGet may be upgraded by simply running the latest installer.

Rollback

See restoring your ProGet instance for how to rollback. Restoring a v4.3 version should not require a database restore since all database changes were additive.
