5.6 is Released!

Note: this is not the latest version of BuildMaster

BuildMaster is an Application Release Automation tool built for DevOps. You can create complex release pipelines all managed from one central dashboard.

If you're upgrading, make sure to check out Upgrade Notes for BuildMaster 5.6.

Issue Tracking Integration

BuildMaster now has a first-class integration with a variety of Issue Trackers. Combining your issue tracking with BuildMaster allows for increased visibility across your application development process. It provides insight into Testing, QA, Bug status, etc. and allows issues to be transitioned with automation removing unnecessary manual steps.

Issue tracking sources are added by installing the extension for the issue tracker, and adding an Issue tracking source and corresponding resource credential in your instance.

Improved Source Control

Deprecating the "provider model" in favor of using Resource Credentials and Global Variables for configuration, as well as tool-specific Operations. So instead of "Get-Latest using SVN" it would be "SVN-Checkout".
