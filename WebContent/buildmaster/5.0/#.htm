5.0 is Released!

Note: this is not the latest version of BuildMaster

BuildMaster is an Application Release Automation tool built for DevOps. You can create complex release pipelines all managed from one central dashboard. Buildmaster is accessible for the whole organization, and promotes effortless communication between teams and individuals. It was designed from the ground-up to be fast and easy to use for both personal and enterprise use.

We are excited to announce we've officially shipped BuildMaster 5.0. It's a major release of BuildMaster that offers a completely revamped User Interface, a new execution engine, and the introduction of Pipelines as Code. Upgrading? Make sure to check out Upgrade Notes for BuildMaster 5.0

Introducing Pipelines as Code

In Version 5.0, we introduce the concept of "Pipelines as Code" giving users the ability to create and edit pipelines either visually in the UI or via the JSON editor. Adding another layer visibility and making simple or complex changes effortless while maintaining consistency across every release.

A Whole New Look

We've complete revamped BuildMasters Interface, delivering the same clean, modern and intuitive design as our other tools, Otter and ProGet. You can easily navigate to your Releases, Applications, Pipelines, and Plans from any page.

New Execution Engine

As users of Otter already know, this version of the deployment execution engine includes many improvements:

  • Seamlessly transition between text and visual editing modes
  • The ability to call deployment plan templates, i.e. reusable, parameterized plans
  • Write PowerShell or Bash scripts directly in the plan, or reference them from an asset library
  • Extremely powerful and user-friendly asynchrony and deployment parallelism

Get started today!

As usual, we have implemented these main features based on user feedback. If you have a feature you'd like us to add, don't hesitate to contact us using our support channel.
