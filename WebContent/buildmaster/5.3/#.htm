5.3 is Released!

Note: this is not the latest version of BuildMaster

BuildMaster is an Application Release Automation tool built for DevOps. You can create complex release pipelines all managed from one central dashboard. BuildMaster 5.3 is all about usability and extensibility. Now with the Inedo den, you have access to all Inedo integrations and extensions allowing you to link in with all your DevOps tools.

If you're upgrading, make sure to check out Upgrade Notes for BuildMaster 5.3.

Release Templates

Release templates add another level of flexibility in BuildMaster. You are now able to create a release from a template, giving you the ability the make sure the appropriate deployables are added for that release. Vist our documentation to learn more about this added usability.

Extension and Community portal

Adding even more extensibility, we’ve made it even easier to connect with the rest of your DevOps toolchain. You can now not only hook into Otter with infrastructure configuration sync, but connect with the your other tools through extension both supported by Inedo and written by the community, from the Inedo Den.

Have you created your own custom extension using the BuildMaster SDK? Think it's useful for other folks? You can now submit your own extension to the community portal. Become an active member of the inedo community, and help others by sharing your content today!

More Operations. Usability in Mind.

With 5.3 we’ve added even more built-in operations, now with Otter infrastructure configuration sync operation like “Ensure Service” and “Ensure App Pool” provide additional usability. To check out all the new Operation visit the documentation here.

Get started!

You can download BuildMaster and take your own test drive, and see for yourself just how easy expanding your DevOps toolchain can be. If you have a feature you'd like to add, don't hesitate to contact us via the support Q&A forum.
