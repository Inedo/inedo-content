5.1 Overview

Note: this is not the latest version of BuildMaster

BuildMaster is an Application Release Automation tool built for DevOps. You can create complex release pipelines all managed from one central dashboard. Buildmaster is accessible for the whole organization, and promotes effortless communication between teams and individuals. It was designed from the ground-up to be fast and easy to use for both personal and enterprise use.

Upgrading? Make sure to check out Upgrade Notes for BuildMaster 5.1

New Agents for Windows:
More Reliability & Otter-Compatible

This release introduces the Inedo Agent, which is the next generation of agent that's available to use side-by-side with the current agents. These agents use the same communication channel as Otter, which means you only need to install one agent to work with multiple products. The new agents also offer more reliability with poor and interfrastic connections, especially with regards to self-upgrading the agents.

Resource Credentials

The added benefit of resource credentials will simplify connections to external tools and provide greater secret storages. You can now manage, store, and control access to tokens, passwords, API keys, or any other secrets that are needed for automating deployments. Check out the documentation to learn more!

First-Class ProGet Integration

The ProGet integration is now stronger than ever with built-in operations to push and pull packages from a ProGet Universal Feed, and easily deploy them to your environments and servers.
