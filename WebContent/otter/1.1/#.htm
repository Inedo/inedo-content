1.1 is here!

Note: This is not the latest version of Otter.

As the adorable illustration implies, Otter 1.1 introduces support for Linux/SSH-based servers and Git-based configuration storage and rafts! If you're upgrading, make sure to check out Upgrade Notes for Otter 1.1.

Linux Servers / Agent-less Communication

To communicate with Linux servers, Otter uses the lightweight, highly-optimized, and resilient protocol already enabled on nearly every Linux box: SSH and SFTP. You can connect with a private key or username & password resource credential.

Git-based Configuration & / Rafts

Although rafts are an advanced feature, they are an integral part of how Otter works. The good news is, even if you don't need to use multiple rafts, you can simply replacing the built-in raft with a Git-based raft, and now all of your configuration plans, orchestration plans, and assets will be versioned!
