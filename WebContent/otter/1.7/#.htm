Version 1.7

We've added two major features in this version of Otter: Advanced Templating and an Enhanced LDAP and Active Directory integration.

Advanced Templating

You can now use the Apply-Template operation to perform advanced text replacement. Templating is also now part of the SDK, so it can be used by custom extensions as well.

Enhanced LDAP Directory

We've added a slightly different LDAP and Active Directory integration. This updated integration will allow permissions to be granted to users and groups defined in a third-party LDAP directory, or Active Directory domain forest.
