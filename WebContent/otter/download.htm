Otter will install on any supported version of Windows; simply click through the Hub installer to get Otter up and running in minutes. During installation you will select the edition you wish to install (Enterprise or free), or enter an existing license key.

You can upgrade to Otter 2.2.5 from any previous version of Otter. Just run the installer to update your installation; the agents will automatically update when the installation is complete.
