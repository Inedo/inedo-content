Introducing 1.0!

Note: This is not the latest version of Otter.

We've officially launched Otter 1.0!

Thanks to everyone who provided feedback in the design and beta cycles.

With your help, we built an Infrastructure as Code product that's really easy to use and allows everyone to understand how your servers are configured, and how you orchestrate infrastructure deployments.

If you're currently using (or have tried to use) other Infrastructure as Code tools, such as Puppet or Chef-- great! Every tool has its benefits, and you can easily use Otter alongside them, as many of our early adopters are currently doing.

Otter provides a different layer of visibility for your entire IT operation, and we're hearing a lot of feedback from our first users that the visual, drag-and-drop interface and native Microsoft Windows support have been game-changers for them.
