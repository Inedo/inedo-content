Operations

By installing the Chocolatey extension, you are given an additional built-in operation:

  • Collect Chocolatey Packages - Collects the names and versions of Chocolatey packages installed on a server.
  • Ensures Chocolatey Package - Ensures that a Chocolatey packages is installed on a server.
  • Install Chocolatey Package - Installs a Chocolatey package on a server
