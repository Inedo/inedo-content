Operations

Once the VSTS extension (Visual Studio Online and TFS) is installed in Otter, additional built-in operations can be called in a plan, such as:

  • Download Artifact from TFS2015 or VSO - Downloads an artifact from the specified TFS Server or VSO and save it to the artifact library
  • Queue TFS 2015/VSO Build - Queue a build in TFS or VSO optionally waiting for its completion
  • Create TFS Work Item -Creates a work item in TFS
  • Update TFS Work Item - Updates an existing work item in TFS
