Contains actions for integrating with an instance of Jfrog's Artifactory repository

Linking BuildMaster and Artifactory allows you to pull your software components from your Universal Package repository manager.

Specific operations added include:

  • Get Artifact Metadata - Retrieve metadata for an artifact from an Artifactory repository.
  • Promote Build - Change the status of a build, optionally moving or copying the build's artifacts and its dependencies to a target repository and setting properties on promoted artifacts.
  • Retrieve all Build Artifacts - Retrieves an artifact from the specified destination.
  • Upload Artifact - Deploy an artifact to an Artifactory repository.
  • Upload Build - Create a new build in Artifactory.
