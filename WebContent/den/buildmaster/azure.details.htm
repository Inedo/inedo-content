Operations

Legacy Operations

  • Change Deployment Configuration - Updates the configuration of a deployment in Windows Azure.
  • Create Hosted Service -Creates a new cloud service in Windows Azure.
  • Delete Deployment -Deletes a deployment by slot or by name from a cloud service in Windows Azure.
  • Delete Hosted Service -Deletes a cloud service in Windows Azure.
  • Deploy Package -Deploys a Cloud Service package onto Windows Azure.
  • Package Application -Packages Web and Worker Role applications for deployment onto Windows Azure.
  • Swap Deployment -Swaps a production and staging deployment in Windows Azure.
  • Upload Files to Blob Storage -Uploads files from a BuildMaster Server to Windows Azure Blob Storage.
