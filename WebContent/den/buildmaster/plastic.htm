Centralized Release Platform for PlasticSCM

The PlasticSCM extension allows you to utilize PlasticSCM as your Source Control Provider in BuildMaster. The built-in action are:

  • Apply Label - Applies a label to source file.
  • Get Labeled - Gets the version of code from source control with the specified label or tag.
  • Get Latest - Get latest version of code from source control.
