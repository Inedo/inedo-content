What is the Inedo Hub?

The Inedo Hub is a lightweight Windows app used to install, update, and otherwise service applications in the Inedo Suite.

What is the Inedo Hub
Comand-line Interface?

The Inedo Hub includes a command-line utility allowing all product management features to be scripted.

How do I do an Offline Install?

The Inedo Hub can still be used in an offline environment, but it will require some extra configuration.
