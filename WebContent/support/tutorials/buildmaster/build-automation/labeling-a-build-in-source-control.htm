In some source control systems (SVN, CVS, etc.), a label (also known as a "tag" in SVN) is created by creating a fork (or branch) of the code. In others (TFS, Vault, Git, etc.), the operation is simply known as "label".

BuildMaster makes it easy to label code during a build execution. Any source control provider that is configured in BuildMaster can be selected. In this case TFS.

To add a label add the Apply Label operation to your deployment plan.

Note: For Subversion users, make sure to use the Tag operation instead of the Apply Label operation.
