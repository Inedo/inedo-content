4.4 has been released!

Note: This is not the latest version of ProGet.

ProGet is a Universal Package Manager that lets you host, and manage, not only your in-house packages, but also an ever growing list of external repositories including NuGet, npm, Bower, Maven, and PowerShell. It was designed from the ground-up to be fast and easy to use, for both personal, and enterprise use.

Upgrading? Make sure to check out Upgrade Notes for ProGet 4.4

Linux supported

Beginning with Version 4.4 Linux is officially supported.

To learn more about Getting Started with ProGet for Linux, please visit the knowledge base article here.

Introducing Package Promotion

You are now able to promote packages to specific feeds.

This is particularly useful for promoting packages that are production ready, from Development feeds to Productions feeds.

To learn more, visit our documentation on the Package Promotion feature.

Deployment Records

This feature allows you to view additional information about where the packages are deployed, who deployed them, and what application was used for the deployment. The deployment records feature strengthens the connection between ProGet and BuildMaster, and provides useful records for auditing, and compliance requirements.

Want more info? Check out our documentation.
