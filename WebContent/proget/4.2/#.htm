Announcing Amazon S3 and Azure Blob Storage

ProGet now supports Amazon S3 and Azure Blob storage, giving you the ability to store all your packages in the cloud. Utilizing ProGet's cloud package stores provides major benefits, including limitless scalability and easy disaster recovery.

See the Amazon S3 & Azure Blob Storage Documentation for more info.
