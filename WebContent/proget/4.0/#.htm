4.0 is here!

Note: This is not the latest version of ProGet.

ProGet is a universal package manager that lets you host and manger your in-house packages and a growing list of external repositories including NuGet, npm, Bower and Maven. It was designed from the ground-up to be fast and easy to use, for both personal and enterprise use, and includes a source and symbol server to make debugging NuGet Packages a breeze.

ProGet Version 4.0 is all about universal usability. We've done a major redesign of the user interface and added Universal Feeds and Packages. Upgrading? Make sure to check out Upgrade Notes for ProGet 4.0

A Whole New Look

We've complete revamped ProGet's Interface, delivering a clean, modern and intuitive design. You can easily navigate to your Feeds, Packages and Connectors from any page.

Introducing Universal Feeds and Packages

Super simple package management that's actually meant for your applications.

Universal Feeds and Packages give you all the same value and control of packages managers for your applications. Universal Feeds and Packages can be easily extended to serve for more specific package needs, giving you the ability to mange your applications as simple as you develop them.
