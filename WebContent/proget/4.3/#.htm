4.3 has been released!

Note: This is not the latest version of ProGet.

ProGet is a Universal Package Manager that lets you host and manage not only your in-house packages, but also an ever growing list of external repositories (now including PowerShell!). It was designed from the ground-up to be fast and easy to use, for both personal and enterprise use.

Upgrading? Make sure to check out Upgrade Notes for ProGet 4.3

Introducing Docker Feeds

ProGet can now host private Docker repositories. To learn more, visit our Docker Feed tutorial.

Improved Extensibility

We've added custom package filters as extensible components in ProGet, and improved our support for custom user directories.

Our tutorial for ProGet extensibility has been updated to include the new components.

Get started today!
