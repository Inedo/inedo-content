IBM DB2 database software offers industry leading performance, scale, and reliability on your choice of platform from Linux, UNIX and Windows to z/OS.

Integrating DB2 and BuildMaster

This extension includes a database provider which allows BuildMaster's database integration to work with IBM DB2 database software.

In order for this extension to communicate with DB2, the IBM Data Server Driver Package must be installed on the server that will connect to the database (either the BuildMaster server itself, or a server with a BuildMaster agent installed on it).

  • Driver download: http://www-01.ibm.com/support/docview.wss?uid=swg21385217

Note: registration is required on the IBM website in order to download this driver.
