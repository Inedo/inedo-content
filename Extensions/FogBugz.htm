FogBugz is an integrated web-based project management system featuring bug/issue tracking, discussion forums, wikis, customer relationship management, and evidence based scheduling developed by Fog Creek Software. This extension includes a Issue Tracking Provider for FogBugz.

Using the FogBugz Extension

This FogBugz Issue Tracking Provider contained within this extension uses the FogBugz web-based API to provide standard BuildMaster operations of changing issue statuses, appendind issue descriptions, and closing issues. Please refer to the Integrating FogBugz with BuildMaster knowledge base article for more information.

About FogBugz

FogBugz incorporates the lessons Joel Spolsky and the team at Fog Creek have learned over a decade of learning how to write software better. Here are some of the key features from the FogBugz website:

  • Issue and Bug Tracking - Effortless Bug Reporting, Customizable Case Lists, and Full Case Histories
  • Project Management - Wikis, Evidence-based Scheduling, Task Outlining, and Reporting
  • Customer Support - Email Management, AutoSort, and Snippets
  • Advanced Integration - Plugin Development, Source Control, and 3rd-Party Extras
