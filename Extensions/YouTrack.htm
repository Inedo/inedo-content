YouTrack is an agile issue tracking and project tracking tool.

Using the YouTrack Extension

The YouTrack extension contains an issue tracking provider which allows BuildMaster to query a YouTrack instance for the issues associated with each release. Additionally, this extension allows BuildMaster to:

  • Block promotion until all YouTrack issues are resolved
  • Change status of YouTrack issues with a deployment plan action
  • Write a comment to YouTrack issues with a deployment plan action
