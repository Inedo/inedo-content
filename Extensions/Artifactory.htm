This extension contains actions for automating deployments to Artifactory repositories and managing artifacts on the server.

Using the Artifactory Extension

The Artifactory extension allows you mange repositories and upload and download artifacts on Artifactory servers.

  • Create Directory: Creates a directory in the specified repository.

    Action Fields

    Repository

    The repository to create the directory in.

    Directory

    The full path of the directory(ies) to create.

  • Deploy Artifact: Deploys an artifact from the file system to the given Artifactory repository and directory and optionally can set metadata properties of the item.

    Action Fields

    Artifact

    The path to the artifact to be deployed.

    Properties

    Any metadata properties to be set for the item. The format is one property per line written as name=value.

  • Retrieve Artifact: This action downloads the specified artifact from the given repository and directory to a file system location.

    Action Fields

    Item Name

    The (optional) path and name of the artifact to download.

    Properties

    The optional metadata properties that need to match in order for the artifact to be downloaded.

    Output File

    The location and filename for the downloaded artifact.

  • Set Item Properties: Sets or updates the property metadata of the specified item. The item can be a file or a directory. If the item is a directory then the properties of every item in the directory will be set recursively.

    Action Fields

    Item Name

    The file or directory to update.

    Properties

    Any metadata properties to be set for the item. The format is one property per line written as name=value.

  • Move Item: This action moves the specified file or directory to a new repository and/or location.

    Action Fields

    Source Name

    The file or directory to move.

    Destination Repository

    The repository to move the item to.

    Destination Name

    The directory and/or filename to move the item to.

    Suppress Cross Layout Translation

    Flag to set to suppress cross layout translations.
