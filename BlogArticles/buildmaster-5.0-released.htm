BuildMaster 5.0 is Released!

We are excited to announce we’ve officially shipped BuildMaster 5.0. It’s a major release of BuildMaster that offers a completely revamped User Interface, a new execution engine and the introduction of Pipelines.

Introducing Pipelines as Code

In Version 5.0, we introduce the concept of “Pipelines as Code” giving users the ability to create and edit pipelines either visually in the UI or via the JSON editor. Adding another layer visibility and making simple or complex changes effortless while maintaining consistency across every release.

A Whole New Look

We’ve done a major redesign of the interface making it even more users friendly. We updated the UI to incorporate the same clean, modern and intuitive design as our other tools, Otter and ProGet. You can now easily visualize and navigate to your Releases, Applications, Pipelines, and Plans from any page.

PowerFul Execution Engine

As users of Otter already know, the new execution engine is extremely powerful and contains major improvements for usability including:

  • Seamless transitions between text and visual editing modes
  • Ability to call deployment plans as templates
  • Tight PowerShell or Bash script integration
