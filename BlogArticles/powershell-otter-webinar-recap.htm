Webinar Recap: How to provision Windows servers and manage their configuration with PowerShell DSC and Otter

This month, Solutions Architect Mark Johnson joined us to present How to provision Windows servers and manage their configuration with PowerShell DSC and Otter. The recording of this webinar is now up and available for viewing on YouTube.

A quick recap – In this webinar we covered Inedo's configuration management tool Otter, PowerShell DSC, and the benefits of using the two together, along with the following demos:

  • DSC Resources in Otter
  • DSC Configuration Scripts in Otter
  • Installing a new DSC Module
  • Git Configuration Changes and Promotion

Mark has also prepared a Git repository containing assets that were used in the demos, including:

  • Example OtterScripts
  • Example DSC configurations
  • Example JSON configurations
  • Example PowerShell scripts
  • Readme on how to use all of these within Otter

If you want to follow the webinar demos in writing, you can find these and more here.
