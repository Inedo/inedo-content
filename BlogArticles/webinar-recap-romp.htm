Webinar Recap: A Technical Preview of Romp

During our latest webinar, Inedo Founder and CEO Alex Papadimoulis, did an overview of our newest product Romp, followed by a demo.

Romp is designed to help developers package application components together with infrastructure configuration, thereby keeping machine configuration code, application code, and deployment code in lock-step in the same version control repository.

For the folks that missed out, we’ve recorded the full webinar and published it to our YouTube Channel. You can of course check out the recording in its entirety below.
