Inedo's Latest Project: Code Offsets

We’ve put together our collective design efforts, mashed up with our obsession for nerdy things, and created something awesome for the Development community.

Though our last project, Programming Languages ABC++, was meant for the kiddos in our lives, this next project is for those coworkers, friends and family, or even yourself, who need a little reminder when some less than adequate code starts getting thrown around. Our solution, a quick pat on the back, and a few Code Offsets.

Besides constantly aspiring to create an entertaining way to poke fun at coworkers, we also wanted our next project to really benefit the Development community as a whole. Through working with non-profit groups that help promote good code through education, and investing in open source projects, we finally came up with the concept of Code Offsets. Like carbon offsets, which aim to reduce emissions of carbon dioxide/greenhouse gases to compensate, or offset emissions made elsewhere, our Code Offsets are used to offset the bad code that already exists.

So when you purchase a pack of Code Offsets, the proceeds go towards our partner organization, a non-profit group that helps promote good code through education. We’ve designed nine different bills, each offsetting a select amount of bad code. The bills available range from one line, all the way up to one thousand lines. You know, for that one person who really made a mess of things. They feature some of the most notable characters from the history of computing, the founders of the discipline from Babbage to Hopper. Let us know on twitter which bill is your favorite, or who you would like to see on a bill in the future!

We also especially want to thank our partner organization, TECH CORPS. They are an amazing nonprofit organization dedicated to ensuring K-12 students have equal access to technology programs, skills and resources that enhance early learning and prepare them for college and career.

TECH CORPS develops technology programs and deploys tech-savvy talent to assist K-12 schools and youth-serving organizations across the country.

Thanks for being awesome TECH CORPS, and thanks again for working with us!

So check out Code Offsets, and finally do something about bad code.
